Custos Indiretos: Esses custos devem ser rateados e para isso deve ser definida uma base de rateio

A base de rateio é o fator que vai orientar a distribuição dos custos indiretos.

Deve-se escolher o dado que sirva melhor como base de rateio, obter o coeficiente de rateio, dividindo o total a ser rateado pelo total da base escolhida, e multiplicar este coeficiente a cada um dos valores componentes do total da base.
